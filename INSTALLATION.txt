REQUIREMENTS:
-------------
✓ XAMPP (Apache + MySQL + PHP 7.4+)
✓ Web Browser

INSTALLATION STEPS:
-------------------

1. INSTALL XAMPP
   - Download from: https://www.apachefriends.org/
   - Install on your computer
   - Open XAMPP Control Panel
   - Start "Apache" and "MySQL" modules

2. COPY PROJECT FILES
   - Copy this entire folder to: C:\xampp\htdocs\
   - Rename folder to: login-system
   - Final path: C:\xampp\htdocs\login-system\

3. CREATE DATABASE
   Option A - Using Import:
   - Open browser: http://localhost/phpmyadmin
   - Click "Import" tab
   - Choose file: database/schema.sql
   - Click "Go"
   
   Option B - Manual:
   - Open browser: http://localhost/phpmyadmin
   - Click "New" database
   - Name: login_system
   - Click "SQL" tab
   - Copy contents from database/schema.sql
   - Click "Go"

4. ACCESS THE SYSTEM
   - Open browser
   - Go to: http://localhost/login-system/
   - Login with default credentials below

TROUBLESHOOTING:
----------------

Problem: Database connection error
Solution: 
- Check MySQL is running in XAMPP
- Verify database "login_system" exists
- Check config/database.php settings

Problem: Page not found
Solution:
- Ensure folder is in C:\xampp\htdocs\
- Check Apache is running
- Use correct URL: http://localhost/login-system/

Problem: Cannot login
Solution:
- Use exact credentials (case-sensitive)
- Clear browser cache
- Check database has users table with data
